How to Install on Android Device

  1. Download the APK file using the button above
  2. On your Android device, go to Settings > Security and enable Install from Unknown Sources>/em>.
  3. Click on the APK file and follow the prompts to install.
  4. Once installed, open the Vacation Scheduler app from your app drawer, once loaded you will land on the main login screen.

How to Run on Android Emulator

  1. Open Android Studio and launch your emulator.
  2. Drag and drop the APK file into the emulator window.
  3. Wait for the installation to complete.
  4. Find and open the Vacation Scheduler app in the emulator, you will land on the main login screen.
